# Laravel Project (Local Setup with Sail)

This project is built with Laravel and uses Laravel Sail for local development via Docker.

---

## Requirements

Before starting, make sure you have the following installed on your machine:

- Docker Desktop
- Git

No local PHP, Composer, or database installation is required.

---

## Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd <project-folder>
2. Copy the environment file
cp .env.example .env
You may edit .env if you need to change ports or database settings.
```
